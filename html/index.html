STM32-BALL-BALANCE-GAME Instructions and Setup

Introduction

Welcome to the Ball Game! This interactive game is played on a hardware board with an embedded system. The objective is to collect blue target balls by moving your ball while avoiding the red rectangular boundaries.

Project Setup

Follow these steps to set up your project in the STM32 Cube IDE:

Step 1: Unpack the Zip File

  • Unzip the provided file to a suitable location on your computer.

Step 2: Configure Project Settings

Based on the configurations from lab 1, perform the following steps:

  • Suppress FPU Warnings in main.c:
    • Do not remove the lines to suppress the warnings.
    • Go to Project > Properties > C/C++ Build > Settings > MCU Settings:
      • Set Floating-point unit to None.
      • Set Floating-point ABI to Software implementation.
  • Update Linker Configuration:
    • In Project > Properties > C/C++ Build > Settings > MCU GCC Linker > Libraries, add rdimon.
  • Debugger Settings:
    • Change Debug Probe to ST-Link (OpenOCD) in the Debugger tab.
    • In the Startup tab, under Initialization Commands, add monitor arm semihosting enable.

Step 3: Run the Debugger

  • Run the debugger and proceed past the first breakpoint.
  • You should see all tests passing in the console output.

Game Play

Here's how you can play and enjoy the game:

Objective

  • Collect as many blue target balls as possible.

Movement

  • Tilt the board in the desired direction to move your ball.

Avoiding Boundaries

  • Stay away from the red rectangular boundary.

Game Over

  • The game ends if you touch the red boundary.

Restart Game

  • Press the black button to reset and start a new game.

Conclusion

Have fun playing the ball game and see how many blue balls you can collect!
